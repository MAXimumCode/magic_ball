Игра "Волшебный Шар"
====================

Игра предсказывает будущее! Что бы начать, придумайте вопрос, и запустите программу!

Игра написана на [Ruby](https://www.ruby-lang.org/en/).

Запуск в консоли: `ruby magic_ball.rb`
